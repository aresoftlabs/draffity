# Application icons

Placeholder icons live here. Replace with the final brand assets before
publishing the public alpha.

## Required formats (consumed by `tauri.conf.json`)

| File             | Use                             | Min size                   |
| ---------------- | ------------------------------- | -------------------------- |
| `32x32.png`      | tray, small windows             | 32×32                      |
| `128x128.png`    | Linux desktop entry, in-app     | 128×128                    |
| `128x128@2x.png` | hi-DPI, in-app                  | 256×256                    |
| `icon.png`       | high-resolution master          | 512×512+                   |
| `icon.ico`       | Windows installer (MSI) and exe | multi-resolution preferred |

## Regenerating from a single PNG

If you have a single high-resolution PNG (`source.png`, ≥1024×1024), the
fastest way to regenerate the full set is to use the Tauri CLI:

```bash
pnpm tauri icon path/to/source.png
```

This populates the `icons/` directory with every size and format Tauri
needs across all platforms. The current placeholder set was generated by
PowerShell `System.Drawing` during Phase 0 scaffolding and is intended to
be replaced.

## Splash screen

This MVP does not ship a splash window. If you want one in a future
release, create `apps/desktop/splash.html` and reference it from
`tauri.conf.json` under `app.windows[]` with `visible: false` until the
main window is ready.
